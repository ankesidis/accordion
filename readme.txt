A Simple Accordion Block for Gutenberg Editor.

## Installation

1. **Clone the repository:**

    Inside your Wordpress plugin directory

   ```sh
   git clone https://github.com/ankesidis/accordion.git
   cd accordion

2. **Install dependencies:**
    ```sh
    npm install
    npm run build

3. **Activate the plugin:**

Go to the WordPress admin dashboard, navigate to Plugins, and activate the "Accordion" Plugin.

## Usage

**Add the Accordion Block:**

In the Gutenberg editor, add the "Accordion" block from the block library.

**Add Accordion Items:**

Click "Add Item" to add a new accordion item.
Click the title to expand and collapse the content area.
Use the input field to set the title of the accordion item.
Use the toolbar buttons to move items up, move items down, or remove items.

**Customize Appearance:**

Use the block settings in the sidebar to customize the text color and typography of the accordion descriptions.
